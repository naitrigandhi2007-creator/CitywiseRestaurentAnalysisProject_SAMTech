# 🍽️ City-wise Restaurant Analysis Dashboard

An interactive **Data Analytics Dashboard** built using **Python, Pandas, Matplotlib, Flask, HTML, and CSS** to analyze restaurant data from the Enhanced Zomato Dataset.

The project explores restaurant distribution, customer ratings, pricing trends, cuisines, popularity, and business insights through exploratory data analysis (EDA) and visualizations.

---

## 📌 Project Objectives

- Analyze city-wise restaurant distribution.
- Compare average restaurant ratings across cities.
- Identify the most popular cuisines.
- Analyze restaurant pricing trends.
- Evaluate restaurant popularity using customer votes.
- Explore relationships between ratings, prices, and votes.
- Present insights through an interactive web dashboard.

---

## 🚀 Features

- 📊 Dataset Summary
- 🏙️ Top 10 Cities by Restaurant Count
- ⭐ Average Ratings by City
- 🍜 Most Popular Cuisines
- 💰 Average Price Analysis
- ❤️ Restaurant Popularity Analysis
- 📈 Rating Distribution (Histogram)
- 📉 Price vs Rating (Scatter Plot)

## 📌 Business Insights

- Major metropolitan cities have the highest concentration of restaurants.
- Higher restaurant count does not necessarily correspond to higher average ratings.
- Customer votes strongly influence restaurant popularity.
- Premium pricing does not always guarantee higher ratings.
- Popular cuisines dominate the restaurant market.
- Data-driven insights can support better restaurant business decisions.
